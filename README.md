# 🛒 Quicklist - Lista de Compras

> Projeto desenvolvido como parte do Curso Fullstack da Rocketseat.

O **Quicklist** é uma aplicação web simples, moderna e intuitiva para o gerenciamento de listas de compras da semana. Ele permite que o usuário tenha controle sobre os itens que precisa comprar, facilitando a organização do dia a dia.

## 🔗 Acesse o Projeto

A aplicação está no ar via GitHub Pages!  
👉 https://thiagomarcelinovieira.github.io/listadecompras-rocketseat/

---

## ⚙️ Funcionalidades

- **Adicionar itens:** Insira novos produtos na sua lista de compras de forma rápida.
- **Marcar como concluído:** Dê um *check* nos itens que você já pegou ou comprou.
- **Remover itens:** Exclua produtos da lista clicando no ícone de lixeira.

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído utilizando as tecnologias fundamentais do desenvolvimento web:

- HTML5: Estruturação semântica da página.
- CSS3: Estilização, layout e responsividade.
- JavaScript: Manipulação do DOM e interatividade da lista (adicionar, marcar e deletar).

## 🚀 Como executar o projeto localmente

Se você deseja rodar este projeto na sua própria máquina, siga os passos abaixo:

1. Clone o repositório:
```bash
git clone https://github.com/ThiagoMarcelinoVieira/listadecompras-rocketseat.git
```

2. Acesse a pasta do projeto:
```bash
cd listadecompras-rocketseat
```

3. Abra no navegador:

Abra o arquivo `index.html` com duplo clique ou utilize uma extensão como o Live Server no VS Code.
